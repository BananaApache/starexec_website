# StarExecMiami Website

A modern web frontend for the [StarExec](https://starexec.acorn.miami.edu) solver execution platform, built with Django.

## About

This project is a migration of the StarExec platform's web interface from its original Java/Spring/Tomcat stack to Django. It communicates with the existing StarExec backend exclusively through its REST API, preserving all functionality while providing a faster, more maintainable frontend.

## Features

- Browse and navigate the StarExec space hierarchy
- View job results and solver performance metrics
- Solver × domain matrix view with heatmap visualization
- Create and manage jobs
- Download benchmarks, solvers, and result archives
